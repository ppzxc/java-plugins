# java

Java skills for Claude Code — production-grade Java patterns, code review, TDD, JDK 25 features, and Spring Boot 4 conventions.

## Skills

| Skill | Trigger | Description |
|-------|---------|-------------|
| `java:coder` | Writing, designing, or refactoring Java code | Decision tree-driven coding rules from 10 books + JDK 25 + DDD essentials |
| `java:reviewer` | Reviewing Java code, auditing code quality | AI-optimized review checklist — concurrency safety, null safety, architecture, stability |
| `java:tester` | TDD workflow, writing tests | Red-Green-Refactor, Mock strategy, AssertJ patterns, legacy code testing |
| `java:spring` | Spring Boot 4 code, tests, or review | Constructor injection, ProblemDetail (RFC 9457), REST patterns, Spring test annotations |

## Installation

```bash
/plugin marketplace add ppzxc/java-plugins
/plugin install java
```

## Usage

Skills activate automatically based on context, or invoke directly:

```
/java:coder
/java:reviewer
/java:tester
/java:spring
```
